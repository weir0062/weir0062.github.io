Snake Game Plugin

This Snake game is a demonstration of my skills in utilizing external libraries and plugins. streamline the gameplay mechanics and optimize performance, wrote a custom plugin in C++ that manages all the movement and collision detection in real-time. This involved designing and implementing custom algorithms that suit the requirements of the game. By utilizing this plugin, I was able to create a classic snake game.
